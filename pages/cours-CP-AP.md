# Cours CP et AP

Cette page donne des informations sur la différence entre les deux types de cours: 

* Les cours de Connaissances Professionnelles (CP ou CONPRO).
* Les cours d'Applications Professionnelles (AP ou APPRO).

## Différences de contexte

* **Les cours CP** sont dispensés dans les écoles professionnelles. Ces cours sont les mêmes pour les élèves en école plein-temps comme pour les élèves en voie Duale. Ils ont pour objectif la théorie explicative de la pratique.
* **Les cours AP** sont la formation à la pratique professionnelle. Ils sont donnés en école pour les élèves plein-temps. Pour les élèves Duals, cette partie de la formation se déroule dans l'entreprise formatrice. Ils ont pour objectif l'assimilation, l'application et la pratique.

Chaque cours est lié à plusieurs compétences opérationnelles et à leurs objectifs évaluateurs, qui sont définis dans le Plan de formation. Certaines compétences ne sont enseignées que dans les cours CP (par exemple c2.2. *Établir des infographies*). D'autres ne sont traitées que du côté entreprise / cours AP (par exemple c1.3 *Clarifier la collaboration avec des partenaires du projet*, ou d3.1 *Procéder au débriefing avec le mandant*).

La majorité des compétences sont cependant traitées dans les deux contextes, école et entreprise. Pour une même compétence, même si la formulation est identique, la méthode d'enseignement ne sera cependant pas la même pour un cours CP ou pour un cours AP.

## Différences dans le mode d'enseignement

* **Dans les cours CP**, dans le contexte "école", l'enseignement est orienté vers l'apprentissage de nouvelles notions et la théorie explicative. Les cours comprennent souvent une démonstration par l'enseignant·e, suivie d'une séquence pratique. De nombreuses méthodes pédagogiques peuvent être utilisées. La théorie est généralement accompagnée d'un support de cours servant de référence aux élèves. On évite dans ces cours les projets de longue durée, pour privilégier un processus par étapes. 
* **Dans les cours AP**, le modèle est le travail dans l'entreprise, où l'apprenti·e est formé·e dans le contexte de mandats réels. À l'école, l'enseignant·e met en place des situations permettant l'application, souvent sous forme de projet ou de mandat fictif. Un projet pourra se dérouler sur une série de leçons. L'apprentissage collaboratif par groupes sera souvent utilisé. L'enseignant·e peut cependant être amené à renforcer des notions théoriques, si des lacunes se présentent. 

**Cours Interentreprise:** pour les élèves de voie duale, les cours interentreprises (CIE) complètent la formation à la pratique professionnelle et la formation scolaire. Les contenus des CIE sont définis dans le Plan de formation. Pour les élèves en voie Plein-temps, les contenus des CIE sont soit intégrés dans les cours AP, soit organisés sous la forme de workshops.
