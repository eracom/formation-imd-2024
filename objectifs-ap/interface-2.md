# Descriptif

L'objet de ce cours AP de 4 périodes est la réalisation de projets de design et développement web.

Les élèves ont déjà effectué le cours "Projet Interface 1" au 1er semestre. Ce deuxième cours est davantage orienté vers le _développement complet_ d'un site web. 

Ce cours devrait consacrer environ 50% du temps au design (recherche, wireframe, maquettes) et 50% au développemen (code front-end). Les élèves appliquent les compétences acquises durant le cours EP _[Développement d'interfaces](../objectifs-cp/dev-interfaces-1.html)_ qui a démarré au semestre précédent (classe 4-ans) ou dans le même semestre (classe FPC).

## Liens avec d'autres cours

Lien important avec le cours *Développement d'interfaces*.

Lien possible avec le cours *[Communication Marketing](../objectifs-cp/communication-marketing-2.html)*. Des concepts marketing peuvent orienter le contenu des sites web développés (call-to-action, user-flow...).

Dans ce cadre, le projet devra également traiter les aspects suivants: **établir une offre**, rédiger un cahier des charges pour un site web (objectifs évaluateurs a1.2, a1.3, a2.1, a2.3).

Pour chaque projet, l'élève devrait produire une offre pour le client, un devis, sous une forme choise par l'enseignant·e.

## Projets-type

### 1. Site vitrine

Création d’un **site vitrine**, présentation d’une entreprise (p.ex. fleuriste). Pas de fonctions complexes, pas de *e-commerce*: un site-vitrine classique, simple et efficace. 

Un exemple: le site développé dans le cours [learnfromsteph.dev](https://learnfromsteph.dev/lessons/), par Stephanie Eckles.

### 2. Réalisation du portfolio (classe 4-ans)

Pour la classe 4-ans, ce cours devra permettre la réalisation du **portfolio web** de l'élève, en vue de son stage en entreprise. Au semestre suivant (S5) l'élève fera la recherche de stage, qui se déroule en S6. Ce portfolio peut être codé par l'élève, ou réalisé via une plateforme comme Cargo.

Un support de cours pour le projet portfolio est disponible: [cours-web.ch/portfolio/](https://cours-web.ch/portfolio/)

Pour les élèves FPC, la réalisation du portfolio est planifiée pour le semestre suivant, dans le cours **[Projet interface (site web responsive)](interface-site-responsive-fpc.html)**.
